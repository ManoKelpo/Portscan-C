# Portscan-C
A quick portscanner made in C.
## Usage:
 -To scan a single port:
  `./portscanc ip port` 

 -Scan a range of ports:
  `./portscanc ip StartPort EndPort`

 -Scan all the ports:
  `./portscanc ip`

 -Example: 
  `./portscanc 192.168.0.1 0 5000`

## Compatibility:
Tested in Kali GNU/Linux 2023.3 (Debian).
Doest not work in Windows.
